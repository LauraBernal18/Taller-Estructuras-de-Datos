# Taller - Estructuras de Datos

Integrantes:

- Dana Gomez Manrique
- Hilary Herrera Erazo
- Laura Valentina Bernal Lozada

Este repositorio contiene las soluciones en Python 3 de los problemas propuestos en HackerRank para el taller de estructuras de datos.

## Problemas resueltos

1. Detect Whether a Linked List Contains a Cycle
2. Largest Rectangle

Las soluciones corresponden únicamente a las funciones solicitadas por HackerRank, ya que el resto del código (lectura de datos, creación de estructuras y casos de prueba) es proporcionado automáticamente por la plataforma.
